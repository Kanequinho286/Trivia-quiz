# Quiz Trivia Game

Um jogo de quiz interativo que utiliza a API Open Trivia Database com tradução automática para português.

## Funcionalidades

- ✅ Seleção de dificuldade (Fácil, Médio, Difícil, Qualquer)
- ✅ Mínimo de 5 questões
- ✅ Todas as opções de resposta disponíveis
- ✅ Verificação de respostas (Corretas/Incorretas)
- ✅ Placar final com estatísticas
- ✅ Tradução automática para português
- ✅ Categorias variadas
- ✅ Diferentes tipos de questões
